3.盒模型

元素分类

  • 块级元素  div、p、h1-h6、ul、li等
  • 行内元素  a、span、b、strong、em、i、label等
  • 行内块元素 button、input、select、img等
  • 伪元素   ::before ::after

块级元素

不如就叫它行元素吧

  • 默认情况下，块级元素会新起一行。
  • 一般块级元素可以包含行内元素和其他块级元素。
  • 可以设置宽、高、margin等

行内元素

也叫做内联元素

  • 默认情况下，行内元素不会以新行开始。
  • 一般情况下，行内元素只能包含数据和其他行内元素。
  • 不可以设置宽、高、margin-top、margin-bottom

6.继承、覆盖、优先级

继承  部分css属性可以被继承。比如 font-family font-size，但不是所有元素都“听话”，这取决于元素是否存在浏览器自身属性，比如button、select等。

覆盖  相同的样式名，在相同的优先级下，后者会覆盖前者的样式。

优先级   id > class > tag > *
